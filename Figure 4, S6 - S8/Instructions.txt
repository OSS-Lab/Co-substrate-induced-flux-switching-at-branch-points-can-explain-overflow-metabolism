To make the data, run the "make_scatter.py", "make_scatter_nocosub.py" and "make_scatter_oxidase.py" files in runsims. to avoid naming conflicts, these need to be run in separate python kernels. Then the figure scripts in plotdata can be ran. These can be run in the same python kernel, but I recommend using a separate one from the "make_scatter" files. Making the data takes around 15 minutes per set (so around 1 hour total) using 16 threads and 1000 samples. For the paper figures we used 10000 samples

To look at different parameters, you need to change the values in functions in the make_heatmaps_compartment files in simulation_functions and simulation_functions_nocosub folders. For simulation_functions, the functions are make_params_nov25_allranges_nonfixed and make_params_nov25_allranges_fixed (nonfixed vs fixed branch parameters).
